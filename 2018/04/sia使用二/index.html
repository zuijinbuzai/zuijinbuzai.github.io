sia使用（二）

四月 12, 2018 发布在 sia

本文主要介绍sia数据目录修改，钱包重置，以及如何上传文件到sia。

1.配置sia数据目录

在启动过Sia-UI客户端后，会在系统user目录下生成客户端配置文件。 用文本工具打开config.json文件，可以看到里面是一个json

{
    "homePlugin": "Overview",
    "siad": {
        "path": "E:\\siad.exe",
        "datadir": "E:\\data\\Sia-UI\\sia",    \\这个就是siad同步节点等数据存放目录
        "rpcaddr": ":9981",
        "hostaddr": ":9982",
        "detached": false,
        "address": "127.0.0.1:9980"
    },
    "closeToTray": true,
    "width": 1398,
    "height": 731,
    "x": -1427,
    "y": 91,
    "version": "v1.3.2",
    "defaultSiadPath": "E:\\siad.exe",
    "siac": {
        "path": "E:\\siac.exe"
    },
    "address": null,
    "receiveAddress": "7b3a0cbc4d07246213f3c87d5f0c2684d95d5cca2dbcad5dca03e7322ea7857730eb189b7fcd",
    "receiveAddresses": [
        {
            "description": "",
            "address": "b78cfbae4b94f6b91485b9fb1793882902585c35fe90f735813e2071f74f5f0cbb1c33047a2b"
        },
        {
            "description": "收钱",
            "address": "552942fec977a3b3a8df21d038c8d8682d36c22f97c6ad344e121218ab9dde53f2c876cf1db0"
        }
    ]
}

退出客户端，将datadir目录下的文件剪切到一个新目录，将新目录路径写到datadir字段下，保存config文件。然后再次启动客户端，读写数据就是新目录了。

2.重置钱包

打开上文提到的datadir目录 退出客户端，删除wallet文件夹，重启客户端，切换到tab导航栏，会提示重新新建钱包或加载已有钱包。

注意：没必要删除整个datadir目录，否则启动客户端又要经历漫长等待，重新下载整个节点数据

3.导入其它钱包资产

数据框里输入另一个钱包的seed，就可以将资产转移到当前钱包

4.租用存储空间

- 切换到Files选项卡，这个时候我们还没法直接上传文件。使用存储空间，需要支付一定的费用给空间提供者，点击Create Allowance，弹出费用确认框，输入金额确定。（金额自己评估，可反复调整） - 这个时候时候还是无法使用，注意看我截图上的“32contracts”，默认需要连上20个空间提供者，我们才能传输文件。 - 一个文件默认会分成10份，沉余20份，每个主机只接收文件的30份中的0.1x。所以需要连上30个空间租用者，才能最终完成3x。 - 如果想下载文件，只要连上了10台主机，我们就能将文件下下来。 - 注意：即使一份几百k的文件甚至更小的，都将上传120MB的左右，所以上传会比较慢。需要耐心等待，大点文件相对较好。（这里以后的文章再做展开）
